What is CSS ?

  1. Css stands for Cascading Style Sheets.
  2. Css describe how HTML element are to be displayed on screen , paper,or in other ,media
  3. Css is not necessaryCSS save a lot of work.It can control the layout web page all at once
  4. External stylesheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements1.
  • Internal - by using a style element in the head section.
  • External -by using a link a element to link an external CSS file.
  • The most common way to add CSS ,is to keep the styles in external CSS files.However ,in this tutorial we will use inline and internal styles, because this is easier to demonstrate , and easier for you try it yourself

When comparing CSS class vs ID , the difference is that CSS class applies a style to multiple elements .ID ,on the other hand ,applies a style to one unique element .ID is also special in that you can special URL to link directly to an element and it'used by Javascript

In CSS, selector are used to target a specific element or range of element on a web page .Once an element has been targeted ,a style or set of style can be applied to the element

There is wide range of selector available.Two of the most commonly used are class and ID .Both are used to target element to which a style should be applied
